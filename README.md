# VendorBridge - Dynamic RFQ & Vendor Management

This project is a double-sided marketplace platform connecting street food vendors with raw material suppliers.

## New Features
- **Dynamic RFQ CRUD:** Create, Read, Update, and Delete Request for Quotations (RFQs) dynamically connected to MongoDB.
- **Dynamic Item Lists:** Add and remove line items dynamically on RFQ creation and editing.
- **Vendor Invitation Management:** Invite approved vendors to specific RFQs, view the list of invited vendors, and remove invitations dynamically.
